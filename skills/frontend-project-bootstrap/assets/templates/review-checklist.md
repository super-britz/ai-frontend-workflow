# 前端 Review 清单

## 组件与结构

- 复用了现有组件、hooks/composables、services、types。
- 没有重复实现基础组件。
- 文件边界清晰，页面与业务组件职责分离。
- 没有引入无关依赖或扩大重构范围。

## 设计还原

- 组件映射与设计拆解一致。
- 样式使用 token、主题变量或项目既有模式。
- 响应式、长文案、空数据、错误数据表现正常。
- 没有局部 hack 破坏组件库可维护性。

## 状态与交互

- 覆盖 loading、empty、error、permission、disabled、success。
- 表单校验、提交中、防重复提交、错误提示完整。
- 弹窗/抽屉关闭、重置、回填、确认流程正确。
- 权限控制和路由跳转符合项目模式。

## 测试与验证

- 相关单元、组件、E2E 或截图测试已补充。
- lint、typecheck、test、build 已运行。
- 无法运行的命令有明确说明。
- 没有遗留 mock、console、调试样式或临时分支逻辑。
