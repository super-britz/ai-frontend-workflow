# 组件使用规范

## 优先级

1. 项目现有业务组件
2. 项目现有基础组件封装
3. 官方组件库组件
4. 轻量局部组合
5. 新增组件

## 禁止自造

- 按钮
- 表单项
- 弹窗/抽屉
- 表格/列表
- 分页
- 上传
- 通知/提示
- 菜单/Tab
- Tooltip/Popover
- 图标

## 新增组件条件

新增组件必须满足至少一项：

- 现有组件无法覆盖设计或交互。
- 多个页面会复用。
- 需要隔离复杂业务逻辑。
- 需要统一状态、权限或接口行为。
