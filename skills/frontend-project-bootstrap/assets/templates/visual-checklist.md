## 检查项

- 布局结构与设计一致。
- 间距、对齐、密度符合项目规范。
- 字号、字重、行高符合 token 或既有样式。
- 颜色、边框、圆角、阴影符合主题。
- 图标、按钮、表单、弹窗使用正确组件。
- 文案不溢出、不遮挡、不重叠。
- Hover、focus、active、disabled 状态可见。
- 响应式下核心操作不丢失。
- 键盘操作和基础可访问性可用。
