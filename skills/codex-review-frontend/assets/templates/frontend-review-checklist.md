# 前端专项 Review 清单

## 组件

- 没有绕过组件库。
- 没有重复造已有组件。
- 组件 props、事件、slot、样式符合项目约定。
- 公共组件改动没有破坏既有调用方。

## 接口

- 遵守接口契约。
- 没有硬编码接口路径、状态码、分页结构。
- 类型与接口文档一致。
- mock 没有替代真实契约。

## 状态

- loading
- empty
- error
- permission
- disabled
- success
- submitting

## UI

- 桌面端可用。
- 移动端可用。
- 文案不溢出、不遮挡、不重叠。
- 交互态可见。
- 基础可访问性无明显问题。

## 测试与验收

- 相关测试已补充。
- lint/typecheck/test/build 已运行。
- UI 改动有浏览器或截图验收。
- 无法验证的事项已说明。
