# 视口与状态矩阵

## 默认视口

- 桌面：1440 x 900
- 平板：768 x 1024
- 移动：390 x 844

## 必查状态

- default
- loading
- empty
- error
- permission
- disabled
- success
- long-content

## 必查交互

- hover
- focus
- active
- submit
- reset
- open / close
- filter
- pagination
- upload / download

## 通过标准

- 核心内容可见。
- 核心操作可点击或可通过键盘访问。
- 文案不溢出、不遮挡、不重叠。
- 响应式下不丢失关键操作。
- 状态变化有明确反馈。
