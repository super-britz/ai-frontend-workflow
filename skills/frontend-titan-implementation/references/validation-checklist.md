# 验收清单

完成 Figma 到 Titan 代码实现前，至少检查以下内容。

## 视觉

- 对照 Figma 截图检查布局、对齐、间距、尺寸和分组。
- 检查字体、字号、字重、行高、换行、截断和垂直居中。
- 检查颜色、边框、阴影、透明度、背景和圆角是否符合项目 token 或已做合理归一。
- 确认图片、图标、SVG 和媒体资源都能正确渲染。

## 交互

- 检查 hover、focus、active、disabled、selected、loading、empty、error 等相关状态。
- 检查键盘导航和焦点可见性。
- 检查弹窗、抽屉、popover、dropdown、日期选择器、上传组件等能正常打开和关闭。

## 响应式

- 检查桌面端和移动端宽度。
- 确认没有文字重叠、按钮文字溢出、固定尺寸控件撑开布局等问题。
- 确认表格、筛选区、卡片和表单遵循仓库现有响应式模式。

## 代码质量

- 运行项目中与改动相关且可用的最小检查：lint、typecheck、单元测试、构建或页面冒烟验证。
- 搜索是否有项目规则禁止的硬编码值。
- 搜索是否用 Element Plus 替代了 Titan 已覆盖的组件。
- 保持改动范围只覆盖本次实现所需文件。
